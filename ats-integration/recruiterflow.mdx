## Recruiterflow Integration – Customer Guides

Below you’ll find **two** short, non‑technical guides:

1. **Connecting Recruiterflow to Our Platform** – what you need to do once.
2. **What Happens After You Connect** – how everything works automatically.

---

### Part A. Guide for Connecting Recruiterflow to Our Platform

### 1. What You’ll Need

- A **Recruiterflow** account with admin access.
- A modern web browser (Chrome, Firefox, Edge).

### Generate an API Key in Recruiterflow

1. Log into **Recruiterflow**.
2. Click your **avatar → Settings**.
3. Go to **Integrations → API Keys**.
4. Click **Generate New Key**.
   - Enter the **Label**: `Integration with Asendia`.
5. Copy the generated key and store it in a safe location — you’ll need it in the next step.

👉 **If you don’t see the option to generate an API key**, please contact **Recruiterflow Support** and request them to generate one for you.

### 3. Add the “Asendia AI Interview” Stage

1. In Recruiterflow, go to **Settings** → **Job Settings**.
2. Click **Edit Job Template**.
3. Go to **Interview plan**
4. Name the stage **Asendia AI Interview**.
5. Click **Save** and drag it into the position where you’d like the AI assessment to occur.

\*\*Recommended \*\*Add another stage called **Asendia AI Interview Submission** at the end of your funnel. Our platform moves candidates here automatically after their interview is complete.

### Create Custom Fields in Recruiterflow

1. Log into **Recruiterflow**.
2. Navigate to **Settings → Custom Fields → Job**.
3. Click **\+ Job Field**.
4. Create the following fields:

### 2. Asendia Interview Language

- **Field Type**: Single-select
- **Options**:
  - English
  - French
- **Roles with View/Edit Permission**: Recruiter, Contact

### 3. Asendia Hybrid Interview Question

- **Field Type**: Rich Text
- **Purpose**: Add detailed hybrid interview questions.
- **Roles with View/Edit Permission**: Admin, Recruiter, Contact

### 4. Asendia Hybrid Interview

- **Field Type**: Yes/No
- **Purpose**: Checkbox to indicate if a hybrid interview is required.
- **Roles with View/Edit Permission**: Admin, Recruiter, Contact

![Image(7) Pn](/images/image\(7\).png)

### 5. Connect Recruiterflow in Our Platform

1. Log into **Our Platform** → **Integrations** → **Recruiterflow**.
2. Paste the API key you copied in Step 2.
3. Click **Connect**.
4. A green **Connected successfully** badge confirms everything is set.

![Image(8) Pn](/images/image\(8\).png)

### Customizing the AI Interview Experience

You can personalize how candidates interact with the AI interviewer by configuring **voice, image, and page elements**.

1. Go to https://recruiter.asendia.ai/

### 1. Setting Up Interviewer Voice & Image

1. Go to [**Asendia Dashboard**](https://recruiter.asendia.ai/) → **AI Interviewer → Customize**.
2. In the left menu, click **Interviewer Avatar**.
   - Here you can **choose the interviewer’s appearance** (image/avatar).

This ensures that the AI interviewer reflects your organization’s branding

### 2. Hiding Fields from Candidates

1. Go to **AI Interviewer → Customize → Page Elements**.
2. You’ll see a list of elements shown on the application page (e.g., Resume Upload, Location, Company Name, Department, Seniority Level, Workspace Type, Interview Language).
3. To **hide a field from candidates**, simply **toggle it off**.
   - Example: Switch off _Location_ if you don’t want candidates to see job location details.
4. Click **Preview Changes** to verify how the candidate will see the application form.

## **Summary**

- **Avatar Image** → Set inside **Interviewer Avatar**.
- **Hiding Fields** → Controlled under **Page Elements** with on/off toggles.
